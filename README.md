Dailyaide est une plateforme web développée avec Laravel qui facilite la mise en relation entre des particuliers ayant besoin de services de bricolage et des professionnels ou bricoleurs disponibles pour les réaliser.
L’objectif principal est de proposer un espace simple, fiable et intuitif permettant à un client de trouver rapidement la bonne personne pour des travaux variés : installations, réparations, montage, maintenance, etc.
